# app-weather

Make sure to have python 3 or the project won't work. Dockerize app-weather soon.


## Instructions
1. Clone the repo
```
git clone https://github.com/StarOfTheMoon/app-weather/
cd app-weather
```
2. Launch the app
``` 
python weather.py
```
3. View on your favorite browser
http://127.0.0.1:5000/
